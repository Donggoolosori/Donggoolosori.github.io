## <span style="color:rgba(0,70,230,0.7)">Description</span>
- 간단한 tinder clone 앱
- React의 TinderCard Component를 사용하여 사진을 swipe 하여 넘기기 가능
- MERN (MongoDB, Express, React, Node.js) stack을 사용한 Web app
- Express와 Node js를 사용한 서버 생성
- DB로 MongoDB 사용
- React로 frontend 구성
